# todo-server

## SKILLS
- **Backend**
    - `JDK 11 (JAVA)`
    - `Spring-boot`
- **IDE** : `IntelliJ IDEA`
    - **build tool** : `gradle`

- method : POST, GET, PUT, PATCH, DELETE
